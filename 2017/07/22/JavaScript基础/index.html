JavaScript基础

发表于 2017-07-22

简介

  • 用途：写入HTML输出、对事件作出反应、改变HTML内容、改变HTML图像、改变HTML样式、输入验证等。
    注：
    • JavaScript对大小写敏感；
    • 注释//单行或注释/**/多行；

实现

  • 脚本位于< script >与< /script >之间，可放置在< body >或< head >中，或同时存在于两个部分中；
  • 通常的做法是把函数放入< head >部分中，或者放在页面底部，这样就可以把它们安置到同一处位置，不会干扰页面的内容；
  • 外部JavaScript:通常包含多个网页使用的代码、文件扩展名是.js、在< script >标签的src属性中设置该.js文件；

JS表单验证

  • 必填项目、Email验证、用户名密码验证等；

JS正则表达式

  • RegExp对象用于存储检索模式；
  • RegExp.test()检索指定值返回true或false、RegExp.exec()检索指定值返回值或null、RegExp.compile()可以改变检索模式也可添加或删除第二个参数；

JS HTML DOM(文档对象模式 document)

  • 能够创建动态的HTML；
  • 操作HTML元素：通过document.getElementById()、getElementsByName()、getElementsByTagName()等查找元素；

JS HTML BOM(浏览器对象模型 window)

  • 所有浏览器都支持 window 对象，它表示浏览器窗口；所有 JavaScript 全局对象、函数以及变量均自动成为 window 对象的成员，全局变量是 window 对象的属性，全局函数是 window 对象的方法，甚至 HTML DOM 的 document 也是 window 对象的属性之一；
