JavaScript库

发表于 2017-07-22

库(框架)

  • 广泛：jQuery、Prototype、MooTools;
  • 其他：YUI、Ext JS、Dojo、script.aculo.us、UIZE
  • 注意：使用框架之前要对框架进行测试！！！！！

CDN(内容分发网络)

  • 是包含可分享代码库的服务器网络，把框架库存放在一个通用的位置供每个网页分享；
  • 如需在网页中使用 JavaScript 框架库，只需在 < script > 标签中引用该库即可；

jQuery

  • jQuery引用
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"> </script>
  • jQuery描述
    • 主要的 jQuery 函数是 $() 函数（jQuery 函数）：如果您向该函数传递 DOM 对象，它会返回 jQuery 对象，带有向其添加的 jQuery 功能；
    • jQuery 允许通过 CSS 选择器来选取元素；
    • 有用以取代 window.onload() 的 ready() 方法；
    • 允许链式语法；

Prototype

  • Prototype引用
    <script src="http://ajax.googleapis.com/ajax/libs/prototype/1.7.1.0/prototype.js"> </script>
  • Prototype描述
    • $() 函数接受 HTML DOM 元素的 id 值（或 DOM 元素），并会向 DOM 对象添加新的功能；
    • 会向浏览器及 HTML DOM 添加扩展；
    • 允许链式语法；
