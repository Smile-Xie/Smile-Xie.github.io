2、段落

  • 通过< p >定义；
  • 使用< br />可在不产生新段落的情况下进行换行；
  • 段落的行数依赖于浏览器窗口的大小；

3、样式

  • 直接，将样式添加到元素，使用元素的style属性；
  • 间接，可在独立的CSS文件中定义样式；
    注：应避免使用的标签和样式(用样式代替)有< center >、< font >、< basefont >、< s >、< strike >、< u >、align、bgcolor、color;

4、文本格式化

  • 改变文本的外观和含义：一，基于内容的样式，二，物理样式；
  • 文本格式化标签、计算机输出标签、引用和术语定义标签
    注：长引用< blockquote >插入换行和外边距，短引用< q >添加双引号；

5、CSS

  • 插入样式表的三种方法：

    1. 1.外部样式表(样式应用到多个页面)

    2. 2.内部样式表(单个文件需要特别样式)

    3. 3.内联样式(单个元素需要特别样式)

  • 样式相关标签
    < style >定义样式定义，< link >定义资源引用，< div >定义文档中的节或区域，< span >定义文档中行内的小块或区域；

10、块

  • HTML元素基本可分为块级元素和内联元素；块级元素在浏览器显示时常以新行来开始和结束，内联元素常不会以新行开始；
  • < div >块级元素
    • 常见用途一，定义文档中的分区或节，可用作容器组合其他元素；
    • 常见用途二，与CSS一起使用，可用于对大的内容块设置样式属性；
    • 常见用途三，取代表格布局的老式方法实现文档布局！！
      注：table主要作用是显示表格化的数据；
  • < span >内联元素
    • 常见用途一，定义文档中行内的小块或区域，可用作容器组合其他元素；
    • 常见用途二，与CSS一起使用，可用于对部分文本设置样式属性；

11、类
