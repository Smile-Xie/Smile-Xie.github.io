jQuery简介

发表于 2017-07-23

特性与引用

  • HTML元素选取、HTML 元素操作、CSS 操作、HTML 事件函数、JavaScript 特效和动画、HTML DOM 遍历和修改、AJAX、Utilities；

  • 引用：在自己的计算机上存放 jQuery 库然后直接引用，或从 Google 或 Microsoft 加载 CDN jQuery 核心文件；

语法(为HTML元素选取和执行操作编制)

  • $(selector).action()：$定义jQuery、selector查询元素、action执行操作；
  • 所有jQuery函数位于一个document ready函数中，防止文档未完全就绪运行函数导致操作失败；

示例

selector

  • selector允许通过标签名、属性名或内容查询元素；
  • selector允许查询HTML/DOM元素组或单个元素；
  • 元素selector
    • $(“p”) 选取 < p> 元素；
    • $(“p.intro”) 选取所有 class=”intro” 的 < p> 元素；
    • $(“p#demo”) 选取所有 id=”demo” 的 < p> 元素；
    • $(“ul li:first”) 选取每个 < ul> 的第一个 < li> 元素；
  • 属性selector
    • $(“[href]”) 选取所有带有 href 属性的元素；
    • $(“[href=’#’]”) 选取所有带有 href 值等于 “#” 的元素；
    • $(“[href!=’#’]”) 选取所有带有 href 值不等于 “#” 的元素；
    • $(“[href$=’.jpg’]”) 选取所有 href 值以 “.jpg” 结尾的元素；
    • $(“div#intro.head”) 选取id=”intro” 的 < div> 元素中的所有 class=”head” 的元素；
  • CSS selector
    • $(“p”).css(“background-color”,”red”);
