HTML表单

发表于 2017-07-20

HTML表单(用于搜集不同类型的用户输入)

  • < form >元素
    • < form >元素用于定义HTML表单;
    • HTML表单包含表单元素,不同类型的 input 元素、复选框、单选按钮、提交按钮等等;
    • < form> … < /form >

  • Action属性

    • 指定某个服务器脚本来处理被提交表单；
    • 如果省略action属性，则默认当前页面；

  • Method属性

    • 规定在提交表单时所用的HTTP方法(GET或POST)

    • GET(默认方法)
      表单被动提交，且没有敏感信息；
      表单数据在地址栏可见；
      适合提交少量数据

    • POST
      适合表单正在更新数据时，或包含敏感信息；
      表单数据在地址栏不可见；
  • Name属性
    • 每个输入字段必须设置一个name属性才能正确提交；

  • < fieldset >组合表单数据(< legend >为< fieldset >定义标题)

    • 实例

  • 表单属性列表

    • 实例
属性 　 描述
accept-charset 规定在被提交表单中使用的字符集(默认：页面字符集)
action 规定向何处提交表单的地址(URL)(提交页面)
autocomplete 规定浏览器应该自动完成表单(默认：开启)
enctype 规定被提交数据的编码(默认：url-encoded)
method 规定在提交表单时所用的HTTP方法
name 规定识别表单的名称
novalidate 规定浏览器不验证表单
target 规定action属性中地址的目标(默认：_self)

HTML表单元素

  • < input >元素
    • 文本字段默认宽度字符20个；
    • 根据不同type属性显示不同形态；
    • type类型有：text(定义常规文本输入)、radio(定义单选按钮输入，多选一)、submit(定义向表单处理程序提交表单的按钮，表单处理程序在表单的action属性中指定)；

  • < select >元素(下拉列表)

    • 实例

  • < option >元素

    • < option >元素定义selected中待选择的选项，通常把首个选项显示为被选选项；
    • 可通过添加selected属性来定义预定义选项；
    • 实例

  • < textarea >元素

    • 此元素定义多行输入字段，即文本域；

    • 实例

  • < button >元素

    • 实例

  • HTML5表单元素
    • < datalist >、< keygen >、< output >等；
    • < datalist >为< input >规定预定义选项列表，< input >的list属性必须引用< datalist >的id属性；
    • 实例

HTML< input >元素的输入类型

  • < input type=”text” >(单行文本输入)

    • 实例


      First name:
      Last name:

  • < input type=”password” >(密码输入，字段被掩码处理)

    • 实例


      User name :
      User password:

  • < input type=”submit” >

    • 实例


      First name:
      Last name:

    • submit元素省略了value属性后获得默认文本，实例：

      First name:
      Last name:

  • < input type=”radio” >

    • 实例


      Male
      Female

  • < input type=”checkbox” >(定义复选框，多选多)

    • 实例


      I have a bike
      I have a car
  • HTML5为< input >新增输入类型
    color、date、datetime、datetime-local、email、month、number、range、search、tel、time、url、week

  • < input type=”number” >(其他类型略)

    • 可用于包含数字值的输入字段，对数字做出限制；

桃花源记

发表于 2017-07-20

　　晋太元中，武陵人捕鱼为业。缘溪行，忘路之远近。忽逢桃花林，夹岸数百步，中无杂树，芳草鲜美，落英缤纷。渔人甚异之。复前行，欲穷其林。
　　林尽水源，便得一山，山有小口，仿佛若有光。便舍船，从口入。初极狭，才通人。复行数十步，豁然开朗。土地平旷，屋舍俨然，有良田美池桑竹之属。阡陌交通，鸡犬相闻。其中往来种作，男女衣着，悉如外人。黄发垂髫，并怡然自乐。 见渔人，乃大惊，问所从来。具答之。便要还家，设酒杀鸡作食。村中闻有人，咸来问讯。自云先世避秦时乱，率妻子邑人来此绝境，不复出焉，遂与外人间隔。问今是何世，乃不知有汉，无论魏晋。此人一一为具言所闻，皆叹惋。余人各复延至其家，皆出酒食。停数日，辞去。此中人语云：“不足为外人道也。”
　　既出，得其船，便扶向路，处处志之。及郡下，诣太守，说如此。太守即遣人随其往，寻向所志，遂迷，不复得路。
　　南阳刘子骥，高尚士也，闻之，欣然规往。未果，寻病终，后遂无问津者。
